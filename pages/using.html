  • Debian. В сущности, первое, что прижилось после того, как стала неудобна винда.

    • http://packages.debian.org/ - мордочка для просмотра пакетов

    • http://www.us.debian.org/international/l10n/po/ru Debian – Состояние PO-файлов для ru. Если захочется попомогать с переводом.

    • http://debian-live.alioth.debian.org/ Debian Live Project - дебьянный livecd.

    • Devuan - потенциально интересный форк Debian. Можно поглядывать. http://www.devuan.org/

    • http://www.rarewares.org/debian.php - некоторое количество доп. софта для дебиану. В основном про аудио.

    • http://fvwm.org/doc/unstable/index.html - Fvwm: Documentation

  • kbdd - https://github.com/qnikst/kbdd

  • perl - http://www.pm.org/faq/ - Perl Mongers Frequently Asked Questions. Это про группы перловщиков.

Тексты

  • emacs - легендарный текстовый редактор.

  • org-mode - Главное, чем я в emacs-е пользуюсь. Планировщик, лёгкий язык разметки и… в общем, удобная штука.

  • - словари.

  • fbreader - http://www.fbreader.org/ - читалка.

  • wikidpad - http://wikidpad.python-hosting.com/wiki/ и http://wikidpad.sourceforge.net/ - десктопная вики. Cбоку показывает дерево, при этом совершенно не напрягается, если страница “сама себе бабушка”.

  • Recoll user manual http://www.lesbonscomptes.com/recoll/usermanual/index.html - удобный мне десктопный поиск.

  • unoconv - конвертирует файлы между форматами, которые знает libreoffice. Напоминашка: unoconv -f формат_в_который имя_файла

  • pandoc - конвертирует файлы между форматами, которые знает сам. В меру сил. Напоминашка: pandoc -o имя_выходного_файла имя_файла. Классная штука для конвертирования разной разметки. pandoc -f html -t org --no-wrap filename.html  xclip Сконвертировали, получили в буфер обмена вполне приличный org, c частью непереваренной разметки в блоках #+BEGIN_HTML … #+END_HTML )

  • ack-grep, silversearcher - искалки текста в файлах

  • chmsee - читать chm

  • fbreader - http://www.fbreader.org/ - читать fb2

  • djview4, djvulibre-bin - читать и обрабатывать djvu

  • evince, zathura - читать pdf.

  • antiword, catdoc - конвертировать doc в txt

  • vilistextum - html в txt с линками как сносками внизу

  • convlit - lit в html + картинки и метадата.

  • convmv - перекодировать имена файлов

  • enca и enconv - определятель и конвертер кодировок

  • fortune-mod - выдаёт фортунки по запросу

  • cookietool - утилитки для сортировки, упорядочения, сравнения файлов с фортунками, поиска дублей и т.п.

  • ikiwiki - локальная вики и вики-конвертер, история хранится в git.

Разное

  • Разное

    • Видео - mplayer, avidemux, vlc, mpv и youtube-dl для скачивания

    • http://www.flashgot.net/ - аддончик к айсвизелу, тьфу, файрфоксу. FlashGot - Best Firefox Download Manager Integration

    • ledger - удобная штука для учёта денег. И что существенно, хранит всю инфу в текстовых человекочитаемых и человекоредактируемых файлах. http://ledger-cli.org/

    • psensor - http://wpitchoune.net/psensor - мониторинг температуры. lm-sensors - часть этой же инфы в консольке.

  • Около apt

    • apt-listchanges, apt-rdepends, apt-transport-https и tor, apt-xapian-index.

    • aptitude

    • debtags

  • Скачивалки - aria2, wget, curl

  • Для запуска команд в какой-то момент - at, batch, anacron, fcron. Особенно fcron.

  • Архивы - arj, bzip2, cpio

  • git и окрестности - git, , vcsh, bup, etckeeper, flashbake (https://github.com/commandline/flashbake), giggle

  • Браузеры - iceweasel, chromium, midori

  • Проверки - clamav, cruft-ng

  • Убирание дублей - fdupes(если чуть-чуть и интерактивненько), rdfind (если много и/или автоматом делать хардлинки)

  • Говорилки - gespeaker, espeak, festival

  • Услеживать за вещами - gcstar. Что-то давно не. Не переместить ли в неиспользуемое?

  • Генеалогия - geneweb. Думаю про lifelines.

  • Картинки - gimp, imagemagick gnuplot, graphviz gpicview сonverseen - конвертер

  • Интернетики, скрытонетики, локалнетики - gogoc, miredo i2p, tor, twister wicd smbnetfs, smbtree

  • tkabber - чудесный клиент для сидения в жабберных конфах. Ну, вообще жабберный, но как “типа-аську” я предпочитаю pidgin, а вот для конф именно tkabber удобен дивно и несравненно. Просто меня не хватает на конфы. http://wiki.tkabber.jabe.ru/

  • клиент для блогов - logjam.

  • cuneiform - OCR. Помню смутно, но точно работало, русский распознавало.

  • stardict-tools - конвертация словарей от стардикта

Впрочем, это может быть и недостатком. http://ignorantguru.github.io/spacefm/. Локальное — SpaceFM User’s Manual file://usr/share/doc/spacefm/spacefm-manual-en.html
